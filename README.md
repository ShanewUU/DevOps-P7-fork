# DevOps-P7-25

## Repository description  

This is repository demonstates an exemplar continuous integration (CI) project for a simple Java Library. The project used [Gradle](https://gradle.com/) as the build system and [GitHub Actions](https://github.com/features/actions) the DevOps automation. The project also uses the GitHub action [upload-artifact](https://github.com/actions/upload-artifact) to upload the build artifact to the repository.
